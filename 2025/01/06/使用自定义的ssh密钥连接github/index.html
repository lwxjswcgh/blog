使用自定义的ssh密钥连接github

发表于2025-01-06|更新于2025-01-06
|浏览量:

新建一个ssh密钥

ssh-keygen -t rsa -C "邮箱" -f id_rsa_xx(密钥名)

以id_rsa_test为例，运行后将会生成私钥id_rsa_test和公钥id_rsa_test.pub

在github上添加ssh密钥

进入github，点击右侧的头像，选择settings
然后选择ssh key

点击右侧的绿色按钮New SSH Key
Title随便起一个名字
Key Tpye不用修改

使用type id_rsa_test.pub查看公钥
复制并粘贴到github的Key中

修改配置

进入用户下的.ssh文件夹
创建一个config文件

touch config

编辑config文件

Host github.com
HostName github.com
PreferredAuthentications publickey
IdentityFile C:\Users\用户名\.ssh\密钥名

确认

使用ssh -T git@github.com来验证是否成功
如果显示：

Hi 用户名! You’ve successfully authenticated, but GitHub does not provide shell access.

说明成功
